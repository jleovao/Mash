MASH project

Text file - first line: holds number of options per category.
Categories: marrying,
                   vehicle,
                   living_quarters,
                   pet,
                   occupation,
                   kids,
                  income,
                  school,
                  hobby,
                  weight.

User story:
- execute program
- display starting menu and see if user wants to manually enter information or ask 
  to automatically generate options for each category
- save options to category array manually or automatically
- start game
- display menu and prompt user to press enter to stop counter
- display menu and for each iteration, use the counter to cycle through
  each option and remove option from category.
- Once each category has one option each, display each option.
- Generate story using option
- Prompt user to play again or exit.


Files used:

mash.h    // Mash class specification file
mash.cpp  // Mash class function implementation file
main.cpp  // Client main program
